SCSS Harjoittelua / Training

Moi. Tämän sivun CSS tiedostossa näet todella paljon parannettavaa. Yritä käyttää esiprosessointia (SCSS) parantaaksesi koodia. Käytä ainakin seuraavia:

Hi. Tidy up the messy CSS (why not the HTML too) and make it better with pre-processing (SCSS). You should at least try the following:

  1. Variables
  2. Nesting
  3. Inheritance / @extend
  4. Mixins

Mixins should only be used if they take in an argument, otherwise, you should extend the selector's rules, whether it be a class, id, or placeholder.
